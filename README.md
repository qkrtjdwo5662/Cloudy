# README.md

## **프로젝트 소개**

클라우드 자원 사용 현황과 비용을 실시간으로 시각화하고 최적화 방안을 제시하는 FinOps 대시보드 서비스

## 개발 기간

**시작일**: 2024.10.12

**종료일**: 2024.11.19
